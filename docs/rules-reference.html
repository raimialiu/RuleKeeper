Cross-Language Rules

These rules work across all supported languages using the unified AST.

All Languages C# Python JavaScript TypeScript Java Go

XL-DESIGN-001 Medium

Method Length
Methods should not exceed a maximum number of lines. Long methods are harder to understand and maintain.
Parameters
max_lines Maximum lines per method (default: 50)

XL-DESIGN-002 Medium

Cyclomatic Complexity
Methods should not have high cyclomatic complexity. Complex methods are harder to test and maintain.
Parameters
max_complexity Maximum complexity score (default: 10)

XL-DESIGN-003 Medium

Parameter Count
Methods should not have too many parameters. Consider using a parameter object instead.
Parameters
max_parameters Maximum parameters allowed (default: 5)

XL-NAME-001 Low

Naming Conventions
Checks naming conventions for classes, methods, and variables according to language-specific standards.
Parameters
class_style Naming style for classes (default: pascal)
method_style Naming style for methods (default: auto)
variable_style Naming style for variables (default: auto)

Python Naming Rules

Python

PY-NAME-001 Low

PEP 8 Naming Conventions
Enforces Python PEP 8 naming conventions: snake_case for functions and variables, PascalCase for classes, UPPER_CASE for constants.
Parameters
allow_private_underscore Allow single leading underscore for private (default: true)

Python Documentation Rules

Python

PY-DOC-001 Low

Missing Docstring
Public functions and classes should have docstrings describing their purpose, parameters, and return values.
Parameters
require_class_docstring Require docstrings for classes (default: true)
require_function_docstring Require docstrings for functions (default: true)
min_function_length Min lines before requiring docstring (default: 5)
skip_private Skip private functions (default: true)

JS-VAR-001 Medium

Prefer Const/Let
Use const or let instead of var for variable declarations. var has function scope which can lead to unexpected behavior.
Parameters
prefer_const Suggest const as preferred (default: true)
